## ğŸ“– Overview

**UnReal** is a sophisticated Chrome extension that analyzes web content to detect AI-generated text, images, and videos, manipulated media, fake news, and misinformation. It uses a multi-layered analysis approach combining statistical pattern matching, local ML inference, advanced image forensics, and optional AI-powered verification through Google Gemini.

### ğŸ¯ Key Capabilities

- **AI Text Detection** - Identifies content generated by ChatGPT, GPT-4, Claude, and other LLMs
- **AI Image Detection** - Multi-layer analysis to detect DALL-E, Midjourney, Stable Diffusion generated images
- **Deepfake Video Analysis** - Flags potential deepfakes through frame-by-frame ML analysis
- **Manipulated Media** - Detects edited, photoshopped, or doctored content  
- **Fake News Detection** - Identifies clickbait, conspiracy theories, and misinformation
- **Health & Financial Scams** - Warns about miracle cures and get-rich-quick schemes
- **Social Media Scanner** - Real-time scanning on Twitter/X, Facebook, Instagram, LinkedIn, Reddit, TikTok

## ğŸš€ Quick Start

### Installation

1. **Clone or download this repository**
   ```bash
   git clone https://github.com/Rachit-Ingole/unreal-chrome-extension.git
   cd unreal-chrome-extension
   ```

2. **Load in Chrome**
   - Open Chrome and navigate to `chrome://extensions/`
   - Enable **Developer mode** (top-right toggle)
   - Click **Load unpacked**
   - Select the `extension` folder
   - Pin the extension for easy access

3. **Start the ML Backend (Required for full functionality)**
   ```bash
   cd extension/backend
   pip install -r requirements.txt
   python server.py
   ```

4. **Start browsing!** The extension works immediately with pattern-based analysis

### Optional: Enable AI Analysis

For enhanced accuracy using Google Gemini AI:

1. Get a free API key from [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Click the UnReal extension icon
3. Open **Settings** (gear icon)
4. Paste your API key and **Save**
5. Enable "LLM Tie-breaker" toggle

## âœ¨ Features

### ğŸ” Multi-Layer Analysis Engine

#### Text Analysis

**1. Statistical Analysis (Always Active)**
- Pattern matching for AI signatures
- Keyword detection for misinformation markers
- URL reputation checking
- Entropy and perplexity calculations

**2. Segment-Based Analysis**
- Breaks content into logical segments
- Per-segment risk scoring
- Visual highlighting of suspicious sections
- Granular analysis for long-form content

**3. Local ML Detection (Requires Python Backend)**
- RoBERTa-based AI text detection model
- Human-likeness verification to reduce false positives
- Runs locally for privacy - no cloud dependency

**4. AI-Powered Verification (Optional)**
- Google Gemini integration as "tie-breaker"
- Only activates for uncertain cases (score 35-65)
- Provides context-aware final verdict
- Respects your API quota

#### Image Analysis Pipeline

UnReal uses a sophisticated multi-layer image analysis pipeline:

| Layer | Name | Purpose |
|-------|------|---------|
| Layer 0 | **Metadata Analysis** | EXIF data, software signatures, AI watermarks |
| Layer 1 | **Forensic Analysis** | Error Level Analysis (ELA), noise patterns |
| Layer 2 | **Mathematical Analysis** | Frequency domain, texture consistency |
| Layer 3 | **Local ML** | Optional CNN-based classification |
| Layer 4 | **Gemini AI** | Vision model for final verification |

#### Video Analysis

- **Deepfake Detection** - ML-based frame analysis for AI-generated videos
- **Backend Processing** - Server-side frame extraction and analysis
- **User-Triggered** - Click to analyze videos on demand

## ğŸ“Š Performance

### Efficiency Features

- **Caching** - Results cached for 10 minutes
- **Debouncing** - Prevents over-analysis on dynamic pages
- **Lazy Loading** - Modules loaded only when needed
- **Quota Management** - LLM only used for uncertain cases
- **Request Serialization** - ML requests queued to prevent overload

### Resource Usage

- **Memory** - ~5-10 MB per tab
- **CPU** - Minimal impact (runs on page idle)
- **Network** - API calls only when LLM enabled
- **Storage** - <1 MB (settings + cache)

## ğŸ› Troubleshooting

### Badge Not Appearing

- Check if extension is enabled in settings
- Verify page has loaded completely
- Try refreshing the page
- Check console for errors

### ML Backend Not Working

- Ensure Python server is running (`python server.py`)
- Check server is on port 8000
- Verify no firewall blocking localhost

### Low Accuracy

- Enable LLM tie-breaker with valid API key
- Check if domain is in trusted sources list
- Some pages may need segment analysis enabled

### API Key Issues

- Verify key is correct (no extra spaces)
- Check API key is active in Google AI Studio
- Ensure you have available quota
- Try regenerating the key

### High CPU Usage

- Disable segment analysis for better performance
- Disable image analysis if not needed
- Clear cache regularly
